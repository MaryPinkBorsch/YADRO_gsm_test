# Тестовое задание в компанию Ядро

План:

- нужна хэш-таблица для словарей индексов строк в позицию в массиве (в цсв табличке)
- обработать как 2мерный массив структур типа "cell"
- расчеты рекурсия с записью результата и статуса обработки
- обозначение в структуре "целл": есть поле "тип" (инт, чар или в процессе) и само поле значение "val" (число либо exspression) -> в процессе обозначает рекурсию, если в рекурсивном цикле переходя по ссылкам наткнемся снова на статус "в процессе", значит это зацикливание
- обработка ВСЕЕХ ошибок

Ограничения:

- названия колонок только буквы
- максимальное количество колонок 4096
- суммарный размер колонок - мегабайт
- память не очень сильно утекает
